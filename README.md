\* E-Commerce Database System (MySQL)



This project is a MySQL-based database design for an e-commerce application.

The goal of this project is to understand how real-world e-commerce systems manage data such as users, products, orders, and payments using relational databases.



=================================================================



\* Features

\- Well-structured and normalized database schema

\- Use of primary and foreign keys to maintain data integrity

\- Product and category management

\- Order and order item handling

\- Stock management using database triggers

\- Automatic order total calculation using a stored function

\- Cart and wishlist functionality

\- Product reviews and ratings

\- Database views for simplified reporting
